## 六、M2-1 计费服务测试

### 测试脚本（复制到 PowerShell 执行）

```powershell
.\venv\Scripts\python.exe -c "
import sys; sys.path.insert(0, '.')
from datetime import datetime
from backend.services.billing import calculate_fee

tests = [
    (datetime(2026,6,24,10,0), datetime(2026,6,24,10,30), 1, 5.0,  '30分钟'),
    (datetime(2026,6,24,10,0), datetime(2026,6,24,11,0),  1, 5.0,  '1小时'),
    (datetime(2026,6,24,10,0), datetime(2026,6,24,11,10), 2, 8.0,  '1小时10分'),
    (datetime(2026,6,24,10,0), datetime(2026,6,24,13,0),  3, 11.0, '3小时'),
    (datetime(2026,6,24,10,0), datetime(2026,6,24,15,30), 6, 20.0, '5小时30分'),
]

all_ok = True
for entry, exit_t, exp_h, exp_a, desc in tests:
    h, a = calculate_fee(entry, exit_t)
    ok = h == exp_h and a == exp_a
    if not ok: all_ok = False
    print('[' + ('OK' if ok else 'FAIL') + '] ' + desc + ': hours=' + str(h) + ' amount=' + str(a))

print()
print('全部通过' if all_ok else '存在失败')
"
```

### 预期输出

```

[OK] 1小时: hours=1 amount=5.0
[OK] 1小时10分: hours=2 amount=8.0
[OK] 3小时: hours=3 amount=11.0
[OK] 5小时30分: hours=6 amount=20.0

全部通过
```
